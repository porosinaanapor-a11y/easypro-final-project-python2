## 📌 Описание проекта

В этой игре два игрока (или один игрок и компьютер) выбирают по одному из трех возможных предметов (камень, ножницы или бумагу), оба показывают, что они выбрали. 
Победитель в этой игре определяется по трем правилам: камень давит ножницы, ножницы режут бумагу, бумага накрывает камень.

Игра состоит из:
- импорт модуля random
- использование модуля сhoice()
- пользователь вводит одно значение наугад (каменьЮ ножницы, бумага)
- проверка условия выйгрыша, проигрыша
- организации выхода из игры

При запуске программа:
1) спрашивает, **камень,ножницы,бумага**;  
2) сравнивает значения;  
3) камень давит ножницы, ножницы режут бумагу, бумага накрывает камень.
